# VSeth16
 A stack oriented 16 bit fantasy console VM implemented solely with raylib and the standard C library.  
 Currently extremely WIP.  
 Build tested with gcc, cmake and ninja on MSYS2 MINGW64 on Windows 10 64 bit.  
 To build:  
 > cmake -B build  
 > cd build  
 > ninja  

Dependency:
	- xorg-dev (for linux)
